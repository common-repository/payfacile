Payfacile shortcode

== Description ==

Insert customizable Payfacile buttons and widgets into your pages and posts easily.


= Features =

* Add `"shortcode"` button and custom form into post and page editor.
* Customizable "product url"
* 3 ways to display: as "direct button", as "modal button" and as "widget" (iframe)

* Customizable button "text button", "text color", "background color"

* Customizable iframe "height" and "width", "border color" and "border width"


= Credits =

Payfacile

= Shortcode =

Insert from post & page editor (wysiwyg button)

* as button *

* into popup *

`[payfacile type_of=button textbutton=buy_now color_text=#ffffff color_bg=#26a699 win_target=popup url=https://www.payfacile.com/test-wp/s/ml]`


** into new window **
`[payfacile type_of=button textbutton=buy_now color_text=#ffffff color_bg=#26a699 win_target=new url=https://www.payfacile.com/test-wp/s/ml]`


**as iframe**

`[payfacile type_of=iframe url=https://www.payfacile.com/test-wp/s/ml outline_width=1 outline_color=#26a699 iframe_width=600 iframe_height=740]`



== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

1. You can either use the automatic plugin installer or your FTP program to upload unziped `payfacile` directory to your `wp-content/plugins` directory.
1. Activate the plugin through the `Plugins` menu in WordPress

If you have to upgrade manually simply repeat the installation steps and re-enable the plugin.


== Frequently Asked Questions ==

= Can I use shortcode into "widgets" ? (sidebar or footer for exemple) = 

Yes, copy paste your shortcode into a "text widget"
